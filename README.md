# JJ's Tech Shop - Assignment 1

## Project Overview
This is a full-stack web application developed for UTS Assignment 1. It features an interactive tech-shop interface that demonstrates modern web development skills, including a dynamic product display and a secure checkout process leading to a professional CV.

## Technologies Used
- **Frontend:** React.js
- **Backend:** Node.js, Express
- **Database:** MySQL
- **UI/UX:** CSS (Apple-inspired clean design)

## Setup and Installation

### 1. Database Configuration
1. Open MySQL Workbench.
2. Run the provided `database.sql` script located in the root directory to initialize the `portfolio_db` and its tables.

### 2. Running the Backend (Server)
1. Navigate to the `Server` directory: `cd Server`
2. Install dependencies: `npm install`
3. Start the server: `npm start`
*The server should now be running on port 5001.*

### 3. Running the Frontend (Client)
1. Open a new terminal and navigate to the `Client` directory: `cd Client`
2. Install dependencies: `npm install`
3. Launch the application: `npm start`
*The website will be accessible at http://localhost:3000.*
